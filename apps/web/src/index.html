Qué incluye esta base web

🔎 Búsqueda jurídica

Preparada para filtros por tribunal, año, materia, tipo de conflicto y palabras clave.

🧠 Resúmenes accionables

Espacio para incorporar extractos con hechos, ratio decidendi y criterio aplicable.

⚖️ Cobertura temática

Alimentos, cuidado personal, comunicación, filiación, adopción y violencia familiar.

📑 Trazabilidad

Diseño orientado a enlazar fuentes originales y referencias normativas por caso.
